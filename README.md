# RiskMap CRM

CRM de prospecção para validação da startup RiskMap — app de gestão de risco operacional para PMEs.

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Ou suba no GitHub Pages para acessar de qualquer lugar

## Deploy no GitHub Pages

1. Suba este repositório no GitHub
2. Vá em **Settings → Pages**
3. Em "Source", selecione `main` e pasta `/root`
4. Clique em Save — em 2 minutos o CRM estará online

## O que tem no CRM

- 150 leads qualificados em 6 nichos (TI, Saúde, Construção, Restaurantes, Contabilidade, Agências)
- Botão WhatsApp com mensagem pré-preenchida
- Sistema de status por lead (Não contatado → Contatado → Respondeu → Interessado)
- Campo de comentários com histórico por lead
- Filtros por nicho, status e busca livre
- Export CSV
- Métricas de funil em tempo real

## Fundadores

- Lindo — Produto, Design, Vendas
- Camila — Dados, Risco, Operações

## Stack

HTML + CSS + JS puro. Zero dependências. Abre em qualquer navegador.
